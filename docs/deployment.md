## Steps to Deploy
1. Clone the repository:
   ```bash
   git clone <repo-url>
   cd tech-dashboard
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set environment variables:
   - Create a `.env` file with API keys and database credentials.

4. Start the server:
   ```bash
   npm run dev
   ```

## Hosting
- Deploy on **Vercel** for serverless execution and cron job scheduling.
